Introduction

That project uses Bootstrap as a basis style. So, all forms structures should follow the Boostrap's recommendations.

Any doubt you can take a look at the Bootstrap Forms documentation.

sr-only

Important note: Remember to always use the label element in order to keep our application well indexable and accessible. You can do that using the sr-only class of bootstrap. e.g:

<form>
  <div class="form-group">
    <label class="sr-only" for="my-sr-only-field">SR Field</label>
    <input type="text" placeholder="SR Field" class="form-control" id="my-sr-only-field" />
  </div>
  <button type="submit" class="cc-button-primary">Submit</button>
</form>

Select fields

All selects have a JS to customize them better, but only the classes and elements are being used in the project.

So, it needs a container element called control and picker, it should be the select container and the select element itself.

<form>
  <div class="form-group">
    <label class="sr-only" for="select-example">Select Example</label>
    <div class="control picker">
      <select class="form-control" id="select-example">
        <option>First Option</option>
        <option>Second Option</option>
        <option>Third Option</option>
      </select>
    </div>
  </div>
</form>
