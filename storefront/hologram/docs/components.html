Buttons

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

Lorem Ipsum 
<a class="cc-button-primary" href="#">
  Lorem Ipsum
</a>
<button class="cc-button-primary">
  Lorem Ipsum
</button>

The button transparent is class which you can add to any other buttons types. It's a special type and will invert the default button style and set a transparent background.

Buttons with Icons

If you want to use a large button with an icon, you will have to add a container to your text with a classe cc-icon-text. See some examples right below.

Try to resize the window and see that this data-td-title will be shown on the left side. So, remember to create this attribute to each td copying the th content.
