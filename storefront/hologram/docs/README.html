Description

This widget Load the NaviStone javascript integration.

NaviStone

Installation and configuration

This widget is global, so it will be available to all pages as soon as you have it installed.

Configs

This widget use one global configuration: * [loadNaviStone] - A flag to enable/disable the load the integration

This widget has one configuration: * [clientID] - The Navstone Client ID

Libraries

oeNaviStone uses:

  • [KnockoutJS] - Javascript UI
  • [jQuery] - DOM Manipulation, Ajax requests...
  • [pubsub] - Standard Oracle topic names list

Apis

No apis have been necessary.
